Karin 事件系统是机器人开发的核心，它捕获各种类型的交互，通过监听这些事件，您可以让机器人对不同情况做出响应。

## 事件基类

事件基类提供所有事件共有的基础属性和方法，所有具体事件类型都继承自事件基类。在开发过程中，您可以依靠这些通用属性快速获取事件的基本信息。

[事件基类](./base.md)

## 消息事件

消息事件触发于接收到用户发送的消息时，包括私聊消息、群聊消息和频道消息等。这是机器人最常用的事件类型，用于实现指令处理、关键词回复和聊天功能。

[消息事件](./message.md)

## 通知事件

通知事件触发于各种状态变更时，例如成员进群退群、群名片变动、消息被撤回等。通过监听这些事件，您可以实现群管理、数据统计等功能。

[通知事件](./notice.md)

## 请求事件

请求事件触发于需要机器人进行操作确认的场景，如好友申请、入群申请或群聊邀请。通过处理这些事件，您可以实现自动化的好友管理和群组管理功能。

[请求事件](./request.md)
