以下是事件的通用属性。

## contact

- 键名：`contact`
- 类型：`Contact`
- 描述：此次事件的来源信息

### 群聊 好友

| 键名  | 类型     | 值               | 描述            |
| ----- | -------- | ---------------- | --------------- |
| scene | `string` | `group` `friend` | `群聊` `好友`   |
| peer  | `string` | -                | `群ID` `好友ID` |

### 文字子频道 频道私聊 群临时会话

::: warning
目前频道场景以`官方QQBot`为主，其他暂不支持  
:::

| 键名    | 类型     | 值                           | 描述                                 |
| ------- | -------- | ---------------------------- | ------------------------------------ |
| scene   | `string` | `guild` `direct` `groupTemp` | `文字子频道` `频道私聊` `群临时会话` |
| peer    | `string` | -                            | `频道ID` `来源频道` `来源群号`       |
| subPeer | `string` | -                            | `子频道ID` `发起方ID`                |
